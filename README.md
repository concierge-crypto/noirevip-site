# NOIREVIP Site (Next.js + Tailwind)

## Quick start
1) Install Node.js 18+
2) In this folder run:
   - npm install
   - npm run dev

## Deploy to Vercel (recommended)
1) Create a Vercel account
2) Import this project
3) Click Deploy
4) Add your domain (noirevip.com) in Project → Settings → Domains
5) Copy DNS records into your domain provider
